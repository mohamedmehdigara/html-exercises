All elements

<a>

The anchor element (<a>) is one of the fundamental elements in HTML.

It allows a user to click on the link text it provides, and navigates to that link's location.

<abbr>

The abbreviation element (<abbr>) is used to mark up abbreviations and acronyms in a document.

<address>

The address element (<address>) is used to provide contact information for the author or owner of a document.
